Linux 应用推荐

2019.02.10

Linux 不仅是命令行强大，GUI也同样精致和高效。

GUI 应用

应用 简介
albert 快速启动工具，类似 Alfred ，支持插件增强
appmenu 全局菜单，mate 和 xfce 可用，参考 macOS 中效果
bleachbit 磁盘清理工具，可删除不需要的语言文件及其他
calibre 电子书管理工具，转换格式、编辑内容很强大
copyQ 剪切板管理工具，支持多标签和固定条目
dbeaver 开源的数据库管理工具，支持的类型超全
deadbeef 音乐播放器，Linux 下的 foorbar2k
timeshift btrfs 快照和 rsync 定时备份工具，比时光机还强大
draw.io 画流程图架构图等等，有桌面版的
enpass 密码管理器，配合浏览器插件很好用，Linux 下高级版免费
firefox 火狐浏览器，container 标签页戳中我痛点
flameshot 最强的截屏工具，没有之一了
gitkraken git图形界面工具，配色很赞
goldendict 离线翻译工具，找好词典很好用，可以发音
gparted 分区工具，磁盘扩缩容图形应用方便
gpick 取色工具，查询颜色RGB值
insomnia REST client工具，主题多，布局合理
meld 文件对比工具
mpv 看不到按钮但配置超多功能很强的播放器
netease-music 网易云音乐的 Linux 版，国内唯一的良心啦
peek 录制 gif，分享某些操作时打字不如图、图不如gif
pencil 原型设计工具，我拿来画极简风格的网页图（不会复杂的
plank dock底栏，参考 macOS 底部的效果
playonlinux Wine 开源前端，很方便
quiterss RSS订阅工具，三栏布局经典好用
redshift-gtk 护眼，类似f.lux，使用 geoclue 获取位置
remmina RDP VNC 远程连接工具，支持 SSH 隧道很赞
screenkey 敲键盘时像字幕一样回显输入，分享时适合使用
seahorse 密码和密钥管理的前端，很方便
telegram 据说隐私保护很好的聊天工具，技术群还蛮多的
tilix 终端，不用 tmux 也能方便的切分终端，参考 macOS 里 iTerm2
typora 所见所得的 markdown 编辑器，表格插入和编辑实在太爽
virtualbox 轻量但功能很强的虚拟机，和 vagrant 很配
visualstudio code 微软出的编辑器，有好多奇怪的插件（大雾
obsstudio 录屏录直播什么的，比 peek 更适合长时间录制，文件很小
wireshark 抓包神器（一直不会用
wps 一般人用来替代 Office 足够了（比如我
zeal 文档查看工具，用了 dash 的资源
zenmap nmap前端，集合了一些常用的参数
zim 维护自己的 Wiki ，我拿来替换印象笔记了
mypaint 有数位板或者触摸屏可以拿来画画
claws mail 邮箱管理，装插件后 systemtray 和通知消息更好用
vlc 跨平台很好用的播放器了
xournal 有数位板或者触摸屏可以拿来写写提纲理理思路，支持导出pdf
zoom 视频会议，分享屏幕，很不错

CLI应用

应用 简介
ansible 批量远程管理主机，个人最爱
aria2c 下载工具，有一堆下度盘的教程
asciinema 终端录屏，适合分享命令行操作
csysdig 高级的系统监控工具
htop top 适合人类使用版
gotop 类似 htop，色彩更鲜艳，晒桌面专用
httpie REST client，高级版curl
iperf 网速测试
jq json 格式化输出
mycli MySQL 管理工具，自动补全很爽
neofetch 查看系统信息
neovim 21世纪的 vim
nmap 端口扫描器
pgcli postgresql 管理工具
shellcheck shell脚本检查
terraform 管理各大云基础设施
yamllint yaml校验
youtube-dl 视频下载，可以下B站
zplug zsh插件配置管理
dive 可以查看 docker 镜像的各个 layer
ripgrep Rust 写的快速检索工具
privoxy 本地代理工具，支持 socks 转 http，支持过滤规则
