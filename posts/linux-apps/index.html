Linux 应用推荐

2019.02.10

Linux 不仅是命令行强大，GUI也同样精致和高效。

GUI 应用

应用 简介
albert 快速启动、查找工具
anki 记忆卡片
appmenu 全局菜单，支持mate、xfce
bleachbit 磁盘清理工具
caffeine 应用全屏时不锁屏
calibre 电子书管理
copyQ 剪切板管理工具
dbeaver 数据库管理
deadbeef 音乐播放器
timeshift 支持 btrfs 快照和 rsync 备份
uget 下载工具
docdevs 文档查看工具，与zeal二选一
draw.io 绘图工具
enpass 密码管理，配合浏览器插件
evolution 邮箱日历联系人套件
firefox 火狐浏览器
flameshot 截图工具
geary 邮箱
gitg git图形界面工具
goldendict 翻译工具
gparted 分区
gpick 取色工具，查询颜色RGB值
insomnia REST client工具，比postman快
mailnag 后台邮件检查
meld 文件对比工具
mpv 世界最强的播放器
netease-music 网易云音乐
ocenaudio 音频编辑器
peek 录制gif
pencil 原型设计工具
plank dock底栏
playonlinux 运行windows程序，wine开源前端
polarr 泼辣修图，收费
postman REST client工具，比 insomnia 慢，耗资源
quiterss RSS订阅
redshift-gtk 护眼
remmina vnc远程
sakura 终端，不依赖桌面，适合i3
screenkey 显示击键，类似字幕
seahorse 密码和密钥管理
telegram 聊天，可以接irc
tilix 终端，支持分屏
typora markdown编辑器
uget 下载工具
virtualbox 虚拟机
visual studio code 编辑器
obs-studio 录屏
wireshark 抓包
wps Office替代
zeal 文档查看工具，与devdocs二选一
zenmap nmap前端
zim 桌面维基
zoom 视频会议
